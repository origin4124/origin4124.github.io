How these pieces relate

1. Start with guidance that helps right away

The dancer and parent lanes are meant to help people understand real training pressure, corrections, recovery, and support more clearly before anything else gets bigger.

2. Let the workbook stay practical and secondary

The workbook is part of this path, but it is not meant to carry the whole thing by itself. It works better when it sits inside a steadier context of useful support and clearer guidance.

3. Grow the broader support path slowly

The larger direction is not hidden, but it is also not being rushed. The goal is to let trust and usefulness come first, then let the broader path become more connected over time.

What this is not

  • Not a finished library yet
  • Not a giant hub with everything built at once
  • Not a hard funnel where every page pushes the same next step

It is a lighter overview layer meant to help people see the current pieces in context.

If you want one clear next step

If you are a dancer, start with the dancer guidance lane. If you are a parent, start with the parent guidance lane. If you want to see one practical support tool already taking shape, the workbook preview is the clearest product-facing next step.
